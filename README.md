# React Emlak Full Stack

React Emlak Full Stack is a modern real estate platform built with React and integrated third-party services.

The application allows users to browse properties, authenticate securely, manage reservations and explore location-based listings through an interactive map system.

The project focuses on full stack frontend architecture, authentication systems, responsive UI development and modern React workflows.

---

# Project Overview

This project was developed to practice modern full stack frontend development using React and external service integrations.

The application includes property listing pages, map-based location visualization, authentication workflows and responsive user interfaces designed for real estate management scenarios.

# Architecture

The project follows a modular React architecture.

- Components are separated into reusable UI structures
- Auth0 is used for authentication and session management
- Leaflet is used for map visualization
- Routing is handled through React application structure
- Services manage frontend data interactions

This structure improves scalability and maintainability.

# Installation

## 1. Clone the Repository

```bash
git clone https://github.com/anilates97/React-EmlakFullStack.git
```

---

## 2. Navigate to the Project Folder

```bash
cd React-EmlakFullStack
```

---

## 3. Install Dependencies

```bash
npm install
```

---

## 4. Start the Development Server

```bash
npm run dev
```
